Angular Lazy Loaded Module Sample Application

The main goal is to demonstrate a lazy loading implementations of angular modules

  • Team Module is pre-loaded, ( {{application.teamCount}} teams found )
    • {{team.name}}
  • User Module is pre-loaded (User state: connected as {{user.displayName}}Anonymous)
  • Player Module is loaded on demand and require user authentification

Key Implementations:

  • ExpressJs REST API server
  • Routes handled with ui-router
  • Lazy Loading handled with ocLazyLoad and FuturState plugins
  • REST resources are handled with Restangular Service
  • Forms generated and validated with JSON schema
  • Growl Notifications
  • Gulp Build process with
    • SASS Compilation
    • JS packaging (concat and uglify)
    • NPM and Bower package dependencies
    • LiveReload (broken)
  • User authentification with Satellizer Library
